# Buyer Checklist

Before using this kit, confirm:

- You can run a TypeScript project locally.
- You understand this is source code, not a hosted SaaS.
- You understand this is not a complete prompt management platform.
- You understand this does not guarantee prompt quality or model behavior.
- You will review and adapt the code before production use.
- You will not store real secrets or private customer data in public prompt fixtures.

Good fit:

- indie AI app builders
- solo SaaS developers
- developers using Codex, Cursor, Claude Code, or similar coding tools
- teams that want a small local prompt versioning starting point

Not a fit:

- no-code users
- buyers expecting a hosted dashboard
- teams needing certified governance, security, privacy, or compliance tooling
- users expecting a Langfuse, PromptLayer, or enterprise prompt platform replacement
