# Framework Notes

This kit is framework-neutral TypeScript.

## Good fit

- Next.js route handlers
- Node.js API routes
- internal AI tools
- simple SaaS backends
- local prompt testing scripts

## Be careful with

- prompts edited by non-technical users
- multi-tenant prompt stores
- regulated workflows
- prompts containing private customer data
- production deployments without review

The included Next.js route file is a reference pattern, not a complete app.
