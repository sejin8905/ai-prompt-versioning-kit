# Production Checklist

Before adapting this kit in production, review:

- Where prompt definitions are stored
- Who can edit prompt definitions
- Whether active version changes require review
- How prompt snapshots are linked to user requests
- Whether prompt values may contain personal or sensitive data
- How rollback is triggered
- How model/provider output is evaluated after a prompt change
- How prompt changes are tested in CI

This kit does not replace security, legal, privacy, or compliance review.
